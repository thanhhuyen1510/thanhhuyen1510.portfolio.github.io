# Portfolio Website

This is a personal **Portfolio Website**, created to showcase background, skills, and projects.  
The site is designed with a modern and minimal style, focusing on clarity and professionalism.

- **About Me**: A short introduction and career goals.  
- **Skills**: Programming languages, tools, and technologies I work with.  
- **Projects**: Highlighted works and personal projects with descriptions.  
- **Contact**: Easy ways to reach me via email or social media.

- (Optional: React, Vue, or Bootstrap if used)

- Build an online presence and personal brand.  
- Provide a central place to share my work and future projects.
